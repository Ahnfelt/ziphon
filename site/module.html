Module: date_iso8601

This module implements ISO 8601 date/time. It is the default calendar for dates and times.

Example code: 

@date (duration)
@date_iso8601
d1 := iso8601_parse("2010-01-01")
d2 := iso8601_parse("2010-03-05")
print("Time between d1 and d2: "..duration(d1, d2))

Exported symbols

Types

This module exports no types.

Values

  • parseDate :: {|String| Date}

    Parses an ISO 8601 date. If the argument is not such a date, it throws a FormatException.

  • interval :: {|Date, Date| Interval}

    Returns the interval between the two given dates.

Module: date

  • Authors: John Lillivan
  • Maintainers: John Lillivan
  • License: Public Domain
  • Branch: release
  • Version: 2.03.01
  • Updated: 2010-03-05

This module is used for representing date/time and for performing arithmetic on them.

Example code: 

@date (parseDate, duration)
d1 := parseDate("2010-01-01")
d2 := parseDate("2010-03-05")
print("Time between d1 and d2: " ++ duration(d1, d2))

Values

  • parseDate :: {|String| Date}

    Parses an ISO 8601 date. If the argument is not such a date, it throws a FormatException.

  • interval :: {|Date, Date| Interval}

    Returns the interval between the two given dates.
